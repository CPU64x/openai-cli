# OPENAPI-CLI

Request openai (ChatGPT) more easily from any terminal.

## Compatibility

| Operating System | Version Needed |
| :---:            | :---:          |
| Windows 10/11    | v1.2           |
| Linux (APT-GET)  | v1.2    |

## Install on 
### Linux
> ./install.sh
### Windows
> ./install.bat

## Launch on
### Linux
> openai "Hello"
### Windows
> openai "hello"
